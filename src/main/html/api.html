API

Query entities:

JSON POST: /api/node/expand

Body:

{ "entity": "f456b14e76e22370a102b93a64d1a927e5dd8cd37963dd3b0829fb9e111b7730", "entityId": 1234560000003, "level": 1, "limit": 100, "existing": [1234560000003,...] }

where

  • entity - contains address, transaction or block hash,
  • entityId - contains internal entity identifier (request should have either entity or entityId).
  • level - recursion level,
  • limit - maximum number of entities to query,
  • existing - array of entities to be excluded from request and also used for additional links retrieval
